¿Qué es Git?

Git es un software de control de versiones gratos y de código abierto. Fue creado por Linus Torvalds en 2005. Esta herramienta es un sistema de control de versiones que fue inicialmente desarrollado para trabajar con varios desarrolladores en el núcleo de Linux.

Esto significa básicamente que Git es un rastreador de contenido, en particular para almacenar código.

Los proyectos de la vida real generalmente tienen múltiples desarrolladores trabajando en paralelo. Así que necesitan un sistema de control de versiones como Git para asegurarse de que no hay conflictos de código entre ellos

Además, lso requerimientos en este tipo de proyectos cambian constantemente. Así que un sistema de control de versiones permite a los desarrolladores revertir y regresar a uan versión anterior de su código.

El sistema de ramas de Git permite a los desarroladores trabajar individualmente en una tarea

Por ejemplo:

  • una rama -> una tarea

  • una rama -> un desarrollador
